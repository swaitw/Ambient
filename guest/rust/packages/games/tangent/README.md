# Tangent

Tangent is an internally-developed game by Ambient; it is a third-person shooter with classes and hovercars, with more coming in the future.

It aims to use as much of Ambient's technology as possible.

## Controls

### On-foot

- **Move**: `W A S D`
- **Jump**: `Space`
- **Sprint**: `Shift`
- **Interact/Enter**: `E`

### Vehicle

- **Move**: `W A S D`
- **Jump**: `Space`
- **Shoot**: `Left-Click`
- **Exit**: `E`
